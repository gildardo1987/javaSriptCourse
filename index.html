comandos git

crear proyecto

git init: esto ocurre luego de llegar por medio de consola git a la ruta matriz del proyecto, y luego das CLICK INIT

agregar componentes a control de git

"git add" + nombre del archivo: luego de estar en la ruta matriz, se da el comando

retirar componentes a control de git

"git rm" + nombre del archivo: luego de estar en la ruta matriz, se da el comando

Guardar version en git

"git commit" + luego de estar en la ruta matriz, se da el comando, aparecera una ventana : registrar descripcion y dar "esc" y luego ":wq"
